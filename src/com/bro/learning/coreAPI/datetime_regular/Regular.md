# 正则表达式( Regular Expression )
一种描述字符串模式的模式语言,由普通字符和特殊字符（元字符）组成,可以：
- 验证字符串是否符合特定格式
- 从文本中提取信息
- 替换字符串中的内容
- 分割字符串

**注意,在Java中,转义字符`\`也需要先转义`\\`**

| 元字符          | 作用                 | 示例                           |
|--------------|--------------------|------------------------------|
| .            | 匹配任意单个字符(除了换行符)    | 	a.c匹配"abc"、"a&c"等           |
| \d           | 匹配数字	              | \d\d匹配"12"、"45"等             |
| \D           | 匹配非数字	             | \D\D匹配"ab"、"cd"等             |
| \w           | 匹配单词字符（字母、数字、下划线）	 | \w+匹配单词                      |
| \W           | 匹配非单词字符(空格、标点等)    | 0                            |
| \s           | 匹配空白字符（空格、制表符、换行等） | 0                            |
| \S           | 匹配非空白字符            | 0                            |
| [abc]        | 匹配括号内的任意一个字符	      | [aeiou]匹配任意元音                |
| [^abc]       | 匹配不在括号内的任意字符	      | [^0-9]匹配非数字                  |
| [a-z]        | 匹配指定范围内的字符	        | [a-zA-Z]匹配所有字母               |
| [a-d[m-p]]   | 并集：匹配a-d或m-p	      | 等价于[a-dm-p]                  |
| [a-z&&[def]] | 交集：匹配d、e或f         | 0                            |
| [a-z&&[^bc]] | 减法：匹配a-z，除了b和c     | 0                            |
| *            | 0次或多次              | 	a*匹配""、"a"、"aa"等            |
| +            | 1次或多次              | 	a+匹配"a"、"aa"等，不匹配""         |
| ?            | 0次或1次              | 	colou?r匹配"color"或"colour"   |
| {n}          | 恰好n次               | 	\d{4}匹配4位数字                 |
| {n,}         | 至少n次               | 	\d{2,}匹配2位或更多数字             |
| {n,m}        | n到m次               | 	\d{3,5}匹配3-5位数字             |
| ^            | 行的开头               | 	^Java匹配以Java开头的行            |
| $            | 行的结尾               | 	end$匹配以end结尾的行              |
| \b           | 单词边界               | 	\bcat\b匹配独立的单词"cat"         |
| \B           | 非单词边界              | 	\Bcat\B匹配"education"中的"cat" |
| ()           | 捕获分组               | 	(abc)+匹配"abc"、"abcabc"      |
| (?:)         | 非捕获分组	             | (?:abc)+匹配但不捕获               |
| (?<name>)    | 命名捕获组              | 	(?<area>\d{3})命名分组为area     |
